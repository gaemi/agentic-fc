# Support

Agentic FC is community-supported.

Use GitHub issues for:

- reproducible bugs,
- documentation gaps,
- feature proposals,
- build or platform problems,
- gameplay balance reports with seed/preset/profile details.

Before opening an issue:

1. Search existing issues.
2. Check [docs/README.md](docs/README.md).
3. Include command output and reproduction steps.

Do not post real Admin or Manager tokens in public issues.
